# Storage

## Overview

{{ user_name }} offers a robust set of storage services designed to cater to a variety of data storage needs. The platform supports multiple storage types, each serving different application scenarios effectively.

| Storage Type       | Description                                                                                   | Use Cases                                         |
|---------------------|-----------------------------------------------------------------------------------------------|--------------------------------------------------|
| **File Storage**     | Perfect for big data analysis and file sharing, file storage offers high performance and scalability. | Big data analysis and collaborative file sharing. |
| **Object Storage**   | Ideal for data backup, archiving, and sharing, object storage provides high availability and reliability. | Data backup, archiving, and sharing of images, videos, and documents. |

## Accessing Storage Services

Navigate to the side navigation bar and click on the "Storage" section to access different types of storage services.


## File Storage

### Step 1: Access File Storage Interface

Navigate to the File Storage interface to manage your files and folders.

![Access File Storage Interface](images/{{ resource_dir_name }}/storage/file_storage_list.png)

### Step 2: Asynchronously Transfer Files from OSS

For transferring files from OSS asynchronously, use the "Async from OSS" feature.

## Object Storage

### Step 1: Access Object Storage Interface

Upon logging into the {{ user_name }} platform, navigate to the Object Storage interface where you can manage your objects.

![Access Object Storage Interface](images/{{ resource_dir_name }}/storage/object_storage_list.png)

### Step 2: View Synchronization List

Monitor the status, progress, and results of all your synchronization tasks from the synchronization list.

![View Synchronization List](images/{{ resource_dir_name }}/storage/object_storage_sync.png)

### Step 3: Rsync Configuration

For synchronizing files, use the "Rsync" feature to mirror your data across different locations.

![Rsync Configuration](images/{{ resource_dir_name }}/storage/object_storage_rsync_config.png){: style="width:30%"}
