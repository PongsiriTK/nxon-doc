# Model Market

## Overview

The Model Market on {{ user_name }} is an integrated platform featuring a variety of pre-trained Large Language Models (LLMs). Users can select suitable models based on their business needs for development, training, tuning, and inference deployment.

### Available Models

The Model Market includes, but is not limited to, the following selected models:

- **meta-llama-3-8b-instruct**: Optimized for a variety of natural language processing tasks.
- **qwen2-72b-instruct**: Ideal for complex reasoning and analytical tasks.
- **mistral-7b-instruct-v03**: Suitable for conversational and interactive applications.

### Usage Steps

#### Step 1: Browse the Model Market
After logging into the {{ user_name }} platform, navigate to the Model Market to view the list of available models.

![Browse Model Market](../images/{{ resource_dir_name }}/modelmarket/model_market_llm.png)

#### Step 2: Select a Model
Filter models based on application scenarios or framework types. The Model Market supports multiple application scenarios, including text, images, natural language processing, and computer vision.

#### Step 3: Deploy a Model
After selecting the desired model, click the deploy button to create an inference service based on the model.

!!! important "Deployment Requirements"
    Review the model's deployment requirements and resource needs before deployment.

#### Step 4: Model Operations
Once the model is deployed, you can perform operations such as viewing API documentation, model fine-tuning, model evaluation, and online experience.

!!! tip "Integration"
    Use the provided API documentation to integrate the model seamlessly into your applications.

![Model Operations](../images/{{ resource_dir_name }}/modelmarket/model_market_details.png)

### Step 5: Search for Models
Use the search function to quickly find models relevant to your needs.

![Search for Models](../images/{{ resource_dir_name }}/modelmarket/model_market_search.png){: style="width:50%"}

## Next Steps

After deploying the model, you can start using it for inference or further development. You can proceed to the [Model Service](modelservice.md) to learn more about managing and optimizing your deployed models.

We wish you a pleasant experience using the Model Market on the {{ user_name }} platform!
