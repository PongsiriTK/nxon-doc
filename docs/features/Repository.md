# Repository

## Overview

The Repository section of {{ user_name }} allows users to manage and organize their machine learning models and Docker images.

## Accessing Repository Section

To access the Repository section, navigate to the side navigation bar and click on the "Repository" section.

## Images

### Step 1: Click on Images

In the Repository section, click on "Images" to manage your Docker images.

### Step 2: Manage Images

Here you can view your images, their artifacts, pulls, and last modified time. You can switch between Public and Private repositories.

### Step 3: Create New Image

Click the "New" button to create a new Docker image repository.

## Models

### Step 1: Click on Models

In the Repository section, click on "Models" to manage your machine learning models.

### Step 2: Manage Models

View your models, their type, tags, summary, path, and operation details.

### Step 3: Create New Model

Click the "New" button to create a new machine learning model repository.
